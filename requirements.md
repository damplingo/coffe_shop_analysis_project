# Требования к аналитическому решению для сети кофеен

## Бизнес-цели

1. Провести ABC-анализ товаров и выявить "мертвый груз" и "топ продаж" в меню для оптимизации ассортиментной матрицы
2. Определить часы и дни, когда каждая кофейня приносит наибольшую и наименьшую выручку, чтобы обоснованно перераспределять персонал и планировать маркетинговые акции

---

## Функциональные требования к дашборду

### 1. Для достижения бизнес-цели 1 (ABC-анализ и оптимизация ассортимента)

1.1. **Автоматическая классификация товаров**  
Дашборд должен автоматически классифицировать все товары на категории A, B, C на основе доли в общей выручке за день/месяц.

1.2. **Визуальное выделение**  
Дашборд должен визуально отделять цветом товары категории C и категории A.

1.3. **Фильтрация по товарной категории**  
Пользователь должен иметь возможность отфильтровать диаграмму ABC-анализа по товарной категории.

1.4. **Дополнительная метрика в таблице**  
Для каждого товара в таблице ABC-анализа дашборд должен дополнительно отображать общее количество проданных единиц за период.

1.5. **Сравнение с предыдущим периодом**  
Дашборд должен автоматически сравнивать количество проданных позиций за текущий период с аналогичным предыдущим периодом (например, неделя к неделе) и подсвечивать товары с отклонением более чем на **N%** для расследования причин.  
> Параметр **N** должен быть настраиваемым.

### 2. Для достижения бизнес-цели 2 (анализ выручки по времени и кофейням)

2.1. **Тепловая карта выручки**  
Дашборд должен отображать тепловую карту выручки для выбранной кофейни:
- Ось **X** — часы работы
- Ось **Y** — дни недели
- Интенсивность цвета — сумма выручки

2.2. **Столбчатая диаграмма чеков по часам**  
Дашборд должен отображать столбчатую диаграмму количества чеков по часам для выбранной кофейни.  
Пользователь должен иметь возможность **переключать отображаемый день недели** прямо на диаграмме.

2.3. **Сводная таблица по кофейням**  
Дашборд должен содержать сводную таблицу со всеми кофейнями и агрегированными метриками:
- Общая выручка за период
- Час с максимальной выручкой
- Час с минимальной выручкой
- Средний чек
